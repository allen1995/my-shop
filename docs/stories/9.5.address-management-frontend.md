# Story 9.5: 前端地址管理页面

## Status
Ready for Review

## Story
As a user,
I want to manage my shipping addresses on the frontend,
so that I can add and edit addresses for orders.

## Acceptance Criteria
1. 地址管理页面已创建

5. 地址选择器已实现（省市区选择）
6. 页面UI美观，符合设计规范
